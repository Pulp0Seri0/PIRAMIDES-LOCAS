# ---FUNCION DEL CODIGO---
Hacer uso de la herramienta: funcion(def), para escribir un codigo que 
permita generar 2 piramides centradas (Superior e Inferior) con el simbolo "#". 
Estas piramides puede variar de tamaño a tu gusto.
***

# ---PARTES DEL CODIGO---
## Funcion: def UP(n):
Es la encargada de generar la piramide superior, empezando con su base hasta terminar en su parte conica.

## Funcion: def DOWN(m):
Es la encargada de generar la piramide inferior, empézando por la parte conica hasta la base.

## Funcion: def PIRAMIDES(n,m):
Es la encargada de juntar las funciones pasadas en una misma y poder generarlas al mismo tiempo dando dos argumentos (n, m).
